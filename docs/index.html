Text Counter

入力内容を送信・保存せず、ブラウザ内で文字数を数えます。

動作方針

  • 入力内容はブラウザ内だけで処理します。
  • 入力内容を外部送信・保存しません。

入力

「見た目上の文字数」は、可能な場合はUnicodeの書記素単位で数えます。

結果

見た目上の文字数
0
空白除外文字数
0
Unicodeコードポイント数
0
JavaScript length
0
UTF-8バイト数
0
行数
0
段落数
0

補足

絵文字、結合文字、異体字セレクタなどを含む場合、 「見た目上の文字数」「Unicodeコードポイント数」「JavaScript length」は一致しないことがあります。

入力内容は送信・保存されません。
